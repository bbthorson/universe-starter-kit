# Subplot Threads — [Book N: Title]

Every thread (main plot, B-plot, character side-arcs, minor recurring) tracked chapter-by-chapter. Catches dropped threads, missing payoffs, and unmotivated reveals.

## Status values

- **`introduced`** — first appearance
- **`developing`** — adding texture or stakes
- **`complicated`** — reversal or escalation
- **`paid off`** — resolved on-page
- **`dropped`** — silently abandoned (this should be rare and intentional; if you mark something dropped, add a `why` note)
- **`carry forward`** — intentionally left open for a future book
