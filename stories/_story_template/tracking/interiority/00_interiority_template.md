# [Character] — Book [N] Interiority

The per-book internal arc. Separate from the canon character profile (which captures who the character IS). This file captures what's HAPPENING INSIDE THIS CHARACTER ACROSS THIS BOOK.

**Length target:** 400–800 words. If it runs longer, it's drifting into canon-profile territory and should be cut.

**Privacy:** interiority docs never leave the repo. Whatever publishing or extraction layer this universe grows, this folder is excluded — it's the author's brain, not content.

---

## Opening state

Where is this character emotionally at the start of the book? What's the baseline anxiety or hunger they're carrying? What contradiction is dormant but pressing?

## Hidden tension

The thing the character is NOT saying or NOT doing this book. The friction between their public/private register and what's actually true underneath. The thing that, if it surfaced, would change everything.

## What cracks open

The specific moment or pressure that forces a register shift, a confession, or a recognition. Cite the chapter and beat. If the crack opens slowly across several chapters, list them.

## What shifts (and what doesn't)

By the end of the book:
- What is genuinely different about this character?
- What looks different but is actually the same (performance of growth without the substance)?
- What is still withheld — both from the other characters and from the reader?
